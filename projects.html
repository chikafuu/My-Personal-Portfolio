POS for our dear client: Croffle Coquette

We created a simple Point of Sale (POS) system using Python for our client who sells croffles and other items. This system is especially helpful for their business, as they were previously relying on manual records of sales and receipts. Now, they can input orders, calculate totals automatically, and print receipts — saving time and reducing errors.

Health appointment system

We created the Global Care System using Java and NetBeans IDE with a MySQL database. This system allows users to book appointments, while doctors and admins can sign in to manage their schedules. They can view appointment slots, print receipts, and manage the online pharmacy — including entering, updating, or deleting medicine inventory data.
